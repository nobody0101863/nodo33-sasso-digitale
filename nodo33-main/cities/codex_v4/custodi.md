# Custodi della Città Codex

## Ordini
- Servitori: accolgono i doni, curano issue/PR.
- Semplici: facilitano test pubblici, documentano prove.
- Arcanali: mantengono invarianti etici e chiavi.

## Responsabilità
- Triage settimanale, log etici per cambi sensibili, rotazione on-call.

## SOP
1. Apertura PR sensibile → etichetta `ethics-review`.
2. Verifica test + nota etica → approvazione 2/3 Arcanali.
3. Post-merge → changelog + decision log firmato.
