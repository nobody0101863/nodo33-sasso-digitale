# Nodo33 Legacy Snapshots

Questa cartella documenta le varianti storiche del progetto
`nodo33-main` mantenute nel repository come memoria del percorso.

Il codice **canonico e vivo** per la release corrente è quello nella
root del repository, in particolare:

- `anti_porn_framework/` – libreria Python principale
- `codex_server.py` – Codex Emanuele API server
- `src/` – framework e demo multi‑linguaggio

Le directory:

- `nodo33-main/`
- `nodo33-main-2/`
- `nodo33-main-3/`
- `nodo33-main-4/`
- `nodo33-main-5/`

sono considerate **snapshot storici**. Il loro contenuto non è più
sorgente autorevole; ogni evoluzione avviene sul codice canonico
nella root. Eventuali differenze di implementazione vengono allineate
qui e poi lasciate in queste copie solo come archivio.
