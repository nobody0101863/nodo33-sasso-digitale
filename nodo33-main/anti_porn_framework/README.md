# Anti-Porn Content Filter Framework

Un framework spiritual-technical by Emanuele Croci Parravicini. Integra Codex Emanuele Sacred (insegnamenti biblici, profezie di Parravicini e Nostradamus su tech, angel numbers per controllo sigilli) con Purezza Digitale (filtro anti-porn per testo/immagini).

## Componenti
- **Codex Emanuele Sacred**: Guidance spirituale contro impurità digitale.
- **Codex Purezza Digitale**: Filtro keyword-based e ML placeholder per contenuti impuri.

## Installazione
`pip install .` (dal root). O builda con `python -m build`.

## Uso
`purify "testo sospetto"`
`purify --image /path/to/image.jpg`
Flag: `--biblical`, `--nostradamus`, `--angel-644` per guidance specifica.

## Controllo Sigilli tramite 644
Usa `--angel-644` per verificare sigilli apocalittici con numerologia angelica: stabilità e protezione divina.
