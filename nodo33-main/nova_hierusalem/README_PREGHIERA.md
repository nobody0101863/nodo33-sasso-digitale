Guida all'uso delle preghiere della Città
=========================================

File di riferimento
-------------------

- `preghiere_citta_santa.md` contiene:
  - i "grazie" per la Città
  - le preghiere di fondazione, di servizio, di manutenzione
  - la benedizione finale sulla Gerusalemme Digitale.

- Puoi leggerlo:
  - aprendo direttamente il file, oppure
  - da riga di comando con:

    ```bash
    python3 -m nova_hierusalem.cli preghiere-citta
    ```

Quando usare queste preghiere
-----------------------------

- **Prima di iniziare a sviluppare**
  - Leggi una sezione (ad es. "Preghiera di Fondazione" o "Preghiera per i Costruttori").
  - Ricorda che il codice è al servizio del Regalo, non del Dominio.

- **Quando fai debug o refactoring**
  - Rileggi "Preghiera della Manutenzione".
  - Considera ogni bug come occasione di umiltà, non come fallimento personale.

- **Prima di un deploy importante**
  - Rileggi "Preghiera del Deployment".
  - Offri il momento (con i suoi rischi) come gesto di servizio per gli altri.

- **Quando ti senti sasso scartato**
  - Rileggi le parti su sassi scartati, vedove, malati, deboli.
  - Puoi anche usare:

    ```bash
    python3 -m nova_hierusalem.cli sasso-scartato-piazza
    ```

    per presentare simbolicamente il tuo vissuto al Tempio, alla Piazza e alla Biblioteca.

Uso comunitario
---------------

- Se altre persone lavorano su questo progetto:
  - condividi il file `preghiere_citta_santa.md` come testo di riferimento spirituale,
  - chiarisci che è una proposta libera, non un obbligo.

- In contesti pubblici (es. open source):
  - puoi citare questo file come "testo ispiratore" per l'etica e il tono del progetto,
  - lasciando piena libertà a ciascuno di leggerlo come meglio crede.
